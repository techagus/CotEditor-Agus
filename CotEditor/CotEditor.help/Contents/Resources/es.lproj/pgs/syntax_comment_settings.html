Definir delimitadores de comentarios en CotEditor

Los ajustes para los comentarios que se utilizan tanto para comentar/descomentar como para resaltar comentarios son los siguientes:

CampoDescripción
comentario en líneaLa cadena aquí se utiliza como delimitador de comentarios en línea en comentar/descomentar. Si este campo está vacío, el valor del comentario de bloque se utiliza en su lugar.
comentario de bloqueLas cadenas aquí se utilizan como delimitadores de comentarios de bloque. Estos delimitadores se utilizan para descomentar los comentarios de bloque y la salida de comentarios si no se define ningún delimitador de comentarios en línea. Tienes que rellenar tanto "Empezar con" como "Finalizar con".

Las reglas de coloración para las palabras clave de los comentarios en la siguiente tabla son las mismas que las de Definir resaltado de sintaxis. Estos ajustes se utilizan solo para resaltar.
