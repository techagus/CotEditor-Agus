Cambiar la configuración de aspecto en CotEditor

En la aplicación CotEditor en tu Mac, utiliza la configuración de Aspecto para elegir la apariencia de los documentos, como el tipo de letra, la altura de la línea y los colores. Para cambiar esta configuración, elija CotEditor > Ajustes y, a continuación, haga clic en Aspecto.

Abre CotEditor para mí

OpciónDescripción
Tipo de letra La fuente y el tamaño de la fuente para el editor. Para cambiar la fuente o el tamaño de la fuente, haga clic en Seleccionar y luego elija una fuente o tamaño de fuente diferente.
Antialising Mostrar texto en el editor con aliasing. Antialising es una función de renderizado que suaviza los bordes irregulares.
Ligaduras Aplique las reglas de ligadura estándar al texto del editor o desactiva todas las ligaduras. Una ligadura es una característica tipográfica que une caracteres específicos como un solo glifo.
Altura de línea Altura de la línea con un valor que indica el número de líneas. El rango debe ser de 0,1 a 10,0.
Cambiar el color de fondo Resalta las líneas en las que existen puntos de inserción con otro color de fondo. El color de la línea actual en el tema seleccionado se utilizará para este resaltado.
Cursor El estilo del cursor en el editor.
Mostrar caracteres invisibles Transforma los caracteres invisibles, como finales de línea, espacios en blanco o caracteres de control, en símbolos alternativos para que sean visibles. VerMostrar caracteres invisibles Para conocer los tipos de caracteres invisibles disponibles.
Mostrar guías de sangría Dibujar guías para cada ancho de sangría en las partes de sangría del documento.
Apariencia Elija la apariencia de la ventana del documento, independientemente del estado de apariencia del sistema.
Tema Elige tu tema de coloración predeterminado para el editor y gestiona los temas. Ver Como personalizar el tema Para la personalización del tema.
Lista de temas Elige tu tema predeterminado para la vista de texto.
Añadir (icono +), Eliminar (icono -) Crea un nuevo tema o elimina el tema seleccionado. Los temas incluidos con CotEditor no se pueden eliminar.
Otras acciones (icono (...))
Renombrar
Cambia el nombre del tema seleccionado, solo si es uno de los creados por tí.
Duplicar
Duplica el tema seleccionado para crear el tuyo propio.
Restaurar
Restaurar el tema seleccionado a su valor predeterminado. Este comando solo está disponible para temas incluidos.
Exportar
Exporta el archivo de tema hecho a partir del tema seleccionado. Con los archivos exportados, los temas para colorear personalizados se pueden usar en otro ordenador. Para usar el archivo exportado, haz Importar.
Compartir
Comparte el archivo de tema hecho a partir del tema seleccionado.
Importar
Esto te permite usar un tema creado en otro ordenador. Después de seleccionar Importar, elija el archivo de tema que desea importar.
Editor de temas Edita el tema seleccionado. Haz clic en los pozos de color para elegir los colores.
Básico
Coloración básica de texto que no sea resaltado de sintaxis. Si la casilla de verificación "Usa el color del sistema" para la "Selección" está activada, la configuración en Ajustes del sistema > Aspecto > Color de resalte se utiliza para el resaltado de la selección.
Sintaxis
Los colores del texto para resaltar la sintaxis.
