Formato de inserción de escritura para Entrega de archivos en CotEditor

CotEditor tiene las siguientes variables para el uso de Entrega de archivos.

Variables para la Entrega de archivos
Cadena preformateadaSerá reemplazado por
<<<ABSOLUTE-PATH>>>La ruta absoluta del archivo entregado (es decir, el archivo de origen).
<<<RELATIVE-PATH>>>La ruta relativa desde su documento hasta el archivo entregado.
<<<FILENAME>>>El nombre del archivo entregado con extensión.
<<<FILENAME-NOSUFFIX>>>El nombre del archivo entregado sin extensión.
<<<FILEEXTENSION>>>La extensión del nombre de archivo entregado.
<<<FILEEXTENSION-LOWER>>>La extensión del nombre de archivo del archivo entregado (en minúsculas).
<<<FILEEXTENSION-UPPER>>>La extensión del nombre de archivo entregado (mayúsculas).
<<<DIRECTORY>>>El directorio del archivo entregado.
<<<FILECONTENT>>>El contenido de texto del archivo entregado (solo funciona cuando se suelta un archivo de texto).
<<<IMAGEWIDTH>>>El ancho de la imagen (funciona solo cuando se suelta un archivo de imagen).
<<<IMAGEHEIGHT>>>La altura de la imagen (solo funciona cuando se suelta un archivo de imagen).

Aquí hay algunos ejemplos de como usar las variables.

ejemplo 1) Cuando dejas caer o entregas foo.html:
Formato de inserción que has establecido<a href="<<<RELATIVE-PATH>>>"></a>
La cadena realmente insertada<a href="foo.html"></a>
ejemplo 2) Cuando dejas caer o entregas foo.jpg:
Formato de inserción que has establecido<img src="<<<FILENAME>>>" width="<<<IMAGEWIDTH>>>" height="<<<IMAGEHEIGHT>>>" />
La cadena realmente insertada<img src="foo.jpg" width="160" height="80" />
