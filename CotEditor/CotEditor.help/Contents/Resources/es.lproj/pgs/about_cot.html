Trabajar con el comando cot

CotEditor tiene 'cot' una herramienta de línea de comandos que le permite controlar CotEditor desde el terminal. Para usar el comando cot, instálalo primero.

Abre CotEditor para mí

Instalar comando cot

Puedes instalar el comando cot a la ubicación deseada mediante la creación de un enlace simbólico en el Terminal. El programa cot es incluye en 'Contents/SharedSupport/bin/cot' de CotEditor.app. A menos que desees personalizar la ubicación, te recomendamos crear el enlace simbólico en '/usr/local/bin/'.

Para crear un enlace simbólico en la ubicación recomendada, ejecute las siguientes líneas en Terminal.

mkdir /usr/local/bin/
ln -s /Applications/CotEditor.app/Contents/SharedSupport/bin/cot /usr/local/bin/cot

Autorización de seguridad

En macOS, se requiere la autorización del usuario para controlar CotEditor desde otras aplicaciones en las que el comando cot se utiliza. La autorización se realiza normalmente a través del cuadro de diálogo que se muestra al usar el comando cot por primera vez. Sin embargo, en algunos casos, es posible que tenga que autorizarlo manualmente desde Ajustes del Sistema > Privacidad y seguridad > Automatización. Selecciona la casilla de verificación para CotEditor (y también System Events para la opción --wait)bajo la solicitud de un cliente, como Terminal.

Instalación de Python 3

El comando cot se implementa en Python 3. Sin embargo, debido a que Python ya no se ha incluido en el sistema desde macOS 12.3, debes instalar python3 manualmente para el comando cot.

Las herramientas de desarrollo proporcionadas por Apple incluyen python3. Ejecute el comando siguiente en el Terminal para instalar las herramientas de desarrollo de la línea de comandos de Apple. Obviamente, cualquier python3 Instalado a través de otros métodos también se puede usar.

xcode-select --install

Desinstalar cot

Para desinstalar el comando cot en la ubicación recomendada arriba a través del Terminal ejecuta:

unlink /usr/local/bin/cot

Usar con opciones

Las siguientes opciones se pueden utilizar en el comando cot.

OpciónDescripción
-w, --waitEspera hasta que se cierre el archivo abierto antes de volver al prompt.
-n, --newCrear un nuevo documento en blanco.
-l, --line <line>Saltar a una línea específica en el documento abierto. <line> es un entero que especifica el número de la línea.
-c, --column <column>Saltar a una columna específica en el documento abierto. <column> Es un entero que especifica el número de la columna.
-b, --backgroundNo poner la aplicación en primer plano.
-h, --helpMostrar ayuda.
-v, --versionInformación de la versión utilizada.

Para las opciones --line y --column, puedes especificar un número negativo para contar desde el final del documento.

Ejemplos

Un comando simple cot para iniciar CotEditor, o simplemente para activarlo si CotEditor ya se está ejecutando.

cot

El siguiente comando abre el archivo foo.txt en CotEditor y mueve el punto de inserción a la línea 200 en el documento abierto. El comando cot no hace nada si el archivo foo.txt no existe.

cot --line 200 foo.txt

El siguiente comando abre el archivo foo.txt en CotEditor y mueve el punto de inserción al final del documento.

cot --line -1 --column -1 foo.txt

También puedes enviar un texto utilizando una tuberia. CotEditor abrirá un nuevo documento con el texto canalizado.

echo "Soy un perro, pero también una vaca al mismo tiempo." | cot

Usando la opción --wait, CotEditor se puede utilizar como editor para git.

git config --global core.editor "cot -w"
