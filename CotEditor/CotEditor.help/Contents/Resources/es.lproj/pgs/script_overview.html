Script CotEditor

CotEditor tiene una función de secuencias de comandos que le permite ejecutar sus propios scripts en CotEditor para manipular documentos o iniciar programas externos. Sus scripts se muestran en el menú de scripts representado con el icono .

Tipos de scripting

En CotEditor, hay dos formas diferentes de escribir scripts:

  • AppleScript
  • UNIX script

Mientras que AppleScript (incl. JXA; JavaScript para Automatización) puede controlar el documento de CotEditor de manera más flexible, los scripts UNIX solo pueden modificar el contenido del documento principal. Sin embargo, en cuanto a los scripts de UNIX, puedes escribirlos en varios idiomas, por ejemplo; Python, Ruby, Perl, etc. Elige un tipo de guión que se adapte a tu uso. Para obtener más información sobre cómo escribir sus guiones, consulte Script CotEditor con AppleScript or Script CotEditor con script UNIX.

Añadir/eliminar scripts

Para añadir o eliminar scripts del menú de scripts, siga las siguientes instrucciones:

  1. Elegir icono script del menú > Abrir la carpeta de scripts y en el Finder se abrirá la carpeta de scripts de CotEditor en:
    ~/Library/Application Scripts/com.coteditor.CotEditor/
  2. Pon tus archivos de script en esta carpeta o elimínalos de allí.
  3. Los cambios que hayas realizado dentro de la carpeta de scripts se reflejarán automáticamente en el menú.

Puede cambiar el orden de los scripts o asignar métodos abreviados de teclado modificando los nombres de sus archivos de script. Ver Personaliza el menú Script para más detalles.
