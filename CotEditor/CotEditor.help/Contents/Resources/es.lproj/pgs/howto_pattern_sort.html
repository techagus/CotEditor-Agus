Ordenar líneas usando un patrón específico en CotEditor

Desde el comando Ordenar por patrón... en el menú Texto > Líneas, puedes ordenar las líneas seleccionadas especificando la forma de comparar.

El simple comando Ordenar en el menú Texto > Líneas es la misma que ordenar "Línea entera" con las opciones "Ignorar mayusculas y minúsculas" y "Respetar las reglas del lenguaje" activadas.

Ejemplo

En este campo, puede comprobar qué parte de una línea se utilizará para la comparación de clasificación.

Ordenar clave

Especifique la parte en las líneas para comparar en la clasificación.

Columna
Trate las líneas como formato CSV (valores separados por caracteres) y utilice la columna específica para la comparación. Los espacios en blanco alrededor del separador se ignorarán.
Delimitador
El carácter que separa cada columna. Puedes usar secuencias de escape como \t para el carácter del tabulador. Se usará una coma cuando este campo esté en blanco.
Posición
El número de la columna que se utilizará como clave de ordenación. El recuento de columnas comienza en 1.
Expresión regular
Especifique la parte para comparar con la expresión regular.
Patrón
El patrón de expresión regular para extraer la parte a comparar.
Ignorar mayúsculas y minúsculas
Se ignoran mayúsculas y minúsculas extrayendo el patrón de expresión regular. Esta opción se utilizará solo para la extracción de la clave de clasificación. Para establecer un estado insensible a mayúsculas y minúsculas para la fase de clasificación, establezca la opción de clasificación a continuación.
Usar grupo capturado
La parte de las coincidencias para usar como clave de clasificación. Toda la coincidencia se utilizará cuando esta casilla de verificación esté desactivada.
Toda la línea
Se comparan líneas enteras.

Opciones de clasificación

Opciones para controlar el comportamiento al comparar las claves de clasificación.

OpciónDescripción
Ignorar mayúsculas y minúsculasIgnorar mayúsculas y minúsculas en la ordenación.
Respeta las reglas del lenguajeClasificar de forma que respeta la configuración actual del idioma. Es decir, ignorar las diferencias triviales, como los anchos de caracteres o las marcas diacríticas.
Tratar los números como valor numéricoLos números dentro de las claves de clasificación se comparan como valor numérico. (ejemplo: archivo12 > archivo5)
Mantén la primera línea en la parte superiorIgnora la primera línea al ordenar.
En orden descendenteOrdenar los resultados en orden descendente.
