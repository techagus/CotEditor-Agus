Define a syntax in CotEditor

CotEditor’s syntaxes control syntax coloring, Outline, and the completion list, all together. To customize syntaxes:

  1. Choose CotEditor > Settings.
  2. Select Format.
  3. Select the syntax you want to customize from the “available syntaxes” list.
  4. Double-click the syntax name or click the Edit button.

To use an existing syntax as a guide for creating your own, choose Duplicate in the action button (gear icon) instead.
