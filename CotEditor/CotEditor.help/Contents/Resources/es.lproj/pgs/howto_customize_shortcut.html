Personaliza los atajos de teclado en CotEditor

Puedes personalizar completamente los atajos de teclado para varios comandos en CotEditor.

El método para cambiar los atajos de teclado depende del tipo de comando:

Métodos para definir atajos de teclado
TipoDescripción
Comando de menú Los métodos abreviados de teclado para los comandos estándar del menú se muestran en este último. Ve al panel Atajos de teclado en Ajustes para cambiar el atajo de teclado de los comandos de menú existentes.
  • Algunos comandos del menú proporcionados por el sistema no se pueden personalizar en CotEditor. Para obtener más información sobre cómo personalizar los atajos de teclado estándar para las aplicaciones CoCoa, consulte la documentación de Apple: Crear atajos de teclado para aplicaciones en Mac
Inserción de fragmentos Atajos de teclado de los comandos para insertar un fragmento de usuario (texto preestablecido). Los comandos que insertan fragmentos también aparecen en el menú, pero los atajos de teclado a estos comandos no se establecen en la configuración de Atajos de teclado, sino en el panel Fragmentos de los Ajustes.
Script de CotEditor Atajos de teclado para ejecutar los scripts de CotEditor del usuario. Incluya una combinación de caracteres específica en el nombre del archivo de script para definir un método abreviado de teclado. Ver Personalizar el menú Script para más detalles.

No puede establecer los atajos de teclado que ya se han tomado (por ejemplo, accesos directos para sus scripts o elementos de menú que no aparecen en la lista). Cuando el mismo atajo de teclado se asigna a varios comandos, solo realiza un comando del conjunto.
