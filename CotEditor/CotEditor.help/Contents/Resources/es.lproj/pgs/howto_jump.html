Go to a specific line in CotEditor

The “Go to Line” command allows you to select and move to a specific line quickly.

  1. Choose Go to Line in the Find menu
  2. Enter the line number where to jump to the dialog.
  3. Click Go, or press Return.

You can also specify a range of lines.

  • If you type in “3”:
    → Line 3 (the third line from the top) will be selected.
  • If you type in “3:2”:
    → What this indicates is “starting from line 3, select 2 lines.” Thus, lines 3 and 4 will be selected.
  • If you type in “3:-1”:
    → What this indicates is “starting from line 3, select the rest lines except the last 1 line.” Thus, if the document has 10 lines, the lines from line 3 to 9 will be selected.
