Cambiar la configuración del formato en CotEditor

En la aplicación CotEditor En su Mac, utilice la configuración de formato para establecer el formato predeterminado para documentos como el tipo de final de línea y la codificación de texto, y para administrar las sintaxis. Para cambiar esta configuración, elija CotEditor > Ajustes y, a continuación, haga clic en Formato.

Abre CotEditor para mí

OpciónDescripción
Finales de línea predeterminados El estilo de final de línea predeterminado que se utilizará al crear un nuevo documento. Esta configuración también se utiliza cuando el documento abierto no incluye ningún final de línea.
Codificación de texto predeterminada La codificación de texto predeterminada que se utilizará al crear un nuevo documento. Ver Manejar varios tipos de finales de línea para más detalles.
Prioridad de las codificaciones La prioridad de las codificaciones y la política de detección de codificación al abrir un archivo existente.
Editar lista...
Haga clic en Editar lista para establecer el orden de las codificaciones que aparecen en el submenú Codificación de archivos en el menú Formato y en la lista desplegable Codificación de archivos en la barra de herramientas. Este orden también se utiliza para la detección de codificación. CotEditor prueba cada codificación desde la parte superior de la lista. Ver Cambiar las prioridades de las codificaciones de texto para más detalles.
Consulte codificación hecha en el documento La declaración de codificación escrita en el documento se respeta al detectar la codificación de texto del documento. Ver Usar declaraciones de codificación para la detección de codificación para más detalles.
Estilo de sintaxis predeterminado Si no hay ninguna sintaxis asociada con la extensión del archivo que abrió, la que especificó aquí se utilizará para colorear.
Esta configuración también se aplica a un documento recién creado.
Estilos de sintaxis disponibles La lista de las sintaxis existentes. Un pequeño punto delante de los nombres de sintaxis representa que está personalizado.
Ver Definir una sintaxis para editar definiciones de sintaxis.
Añadir (icono +), Eliminar (icono -)
Crea una nueva sintaxis o elimina la sintaxis seleccionada. Las sintaxis incluidas con CotEditor no se pueden eliminar.
Editar (icono de lápiz)
Editar la sintaxis seleccionada.
Otras acciones (icono (...))
Duplicar“[nombre de la sintaxis]”
Duplica la sintaxis seleccionada para crear la tuya.
Exportar “[nombre de la sintaxis]”
Exporta el archivo de definición de sintaxis hecho a partir de la sintaxis seleccionada. Puedes usar este archivo exportado para usarlo en otro ordenador. Para usar el archivo exportado, haz Importar.
Compartir
Comparte el archivo de definición de sintaxis hecho a partir de la sintaxis seleccionada.
Importar...
Esto te permite usar una sintaxis creada en otro ordenador. Después de seleccionar Importar, elija el archivo de sintaxis exportado para instalar.
Mostrar conflictos de mapeo de archivos
Compruebe si hay varias sintaxis asociadas con la misma extensión/nombre de archivo.
