Cambiar la configuración de fragmentos en CotEditor

En la aplicación CotEditor en tu Mac, usa la configuración de Snippets para definir textos preestablecidos para insertar con un comando o soltando un archivo en el editor. Para cambiar esta configuración, elija CotEditor > Ajustes, luego haz clic en Fragmentos.

Abre CotEditor para mí

Comando
OpciónDescripción
Definiciones de comandos Defina el comando de inserción de fragmentos. Los comandos definidos aquí se asignarán al menú Texto > Insertar fragmento.
  • Estilo de sintaxis: El fragmento está disponible y aparece en el menú de fragmentos solo cuando un documento que tiene esta sintaxis está en primer plano. Si este campo está vacío (Todos), todos los documentos son elegibles.
  • Nombre: El nombre del comando que se mostrará en el menú de este fragmento.
  • Teclas: El atajo de teclado para realizar este comando de inserción de fragmentos. El atajo de teclado es opcional y se puede dejar en blanco.
Insertar Formato Establezca el texto que se insertará con el comando seleccionado en la lista de definición de comandos anterior.
  • Cuando un fragmento contiene <<<CURSOR>>>, la palabra <<<CURSOR>>> se eliminará en la inserción y se moverá el punto de inserción allí en su lugar.
Entrega de archivos
OpciónDescripción
Definiciones de entrega de archivos Defina las condiciones para realizar la entrega del archivo. Cuando los archivos se insertan directamente en el editor y se cumplen las condiciones definidas aquí, el texto que coincide con los archivos se puede insertar en la ubicación de inserción.
  • Estilo de sintaxis: La configuración se realiza solo cuando un archivo se elimina en un documento que tiene esta sintaxis. Si este campo está vacío (Todos), todos los documentos son elegibles.
  • Extensiones: La configuración se realiza solo cuando se entrega un archivo con una de estas extensiones. Si este campo está vacío (Todos), todos los archivos son elegibles. No se requiere ningún punto para las extensiones de nombre de archivo. Puedes establecer varias extensiones separándolas por una coma.
  • Descripción: Puedes usar este campo para dejar un comentario sobre la definición. Este campo no afecta al comportamiento de la entrega del archivo y se puede dejar en blanco.
Insertar Formato Establezca el texto que se insertará para la condición que se selecciona en la lista de definición de entregar archivos anterior.
  • En un formato de inserción, puedes usar las variables de CotEditor. Las variables se reemplazarán con la información correspondiente del archivo entregado. Ver Formato de inserción de escritura para Entrega de Archivos para los detalles de cada variable.
