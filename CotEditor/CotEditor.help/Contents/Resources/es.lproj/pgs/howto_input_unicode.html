Introducir un carácter usando un código Unicode en CotEditor

Puedes introducir caracteres con códigos Unicode.

Para introducir un carácter con un código Unicode en el formato hexadecimal en la posición de inserción actual:

  1. Elige Edición > Introduce código Unicode...
  2. Introduzca un código en el campo o seleccione un código del historial.
  3. Pulsa la tecla Return.

El código de caracteres Unicode para la entrada debe ser solo de formato hexadecimal, así como código hexadecimal con los prefijos “U+”, “0x”, o “\u”. Por ejemplo, “1F603”, “U+1F603”, “0x1F603”, o “\u1F603” son entradas válidas y equivalentes a “😃”.
